# AGENTS.md — mochapulse

## What this repo IS

A GitHub special repository (profile README). The `README.md` renders on the GitHub profile page. `today.py` queries the GitHub GraphQL API and generates `dark_mode.svg` / `light_mode.svg` embedded in the README.

## Key files and what an agent will miss

- **`today.py`** — single-file Python script. No venv, no pyproject.toml, no poetry. The entire codebase.
- **`dark_mode.svg` / `light_mode.svg`** — BUILT artifacts, never edit these directly. Generated by `today.py`.
- **`cache/requirements.txt`** — the only dependency declaration. Install with `pip install -r cache/requirements.txt`.
- **`cache/2797c90...` (hash-named file)** — runtime GitHub API cache. Gets committed by CI. Do not delete/reformat it.
- **`.github/workflows/build.yaml`** — runs on push to `main` + daily cron (4am UTC). Self-commits generated SVGs via `mochapulse-bot`.

## How to run

```sh
pip install -r cache/requirements.txt
export ACCESS_TOKEN="ghp_..." USER_NAME="mochapulse"
python today.py
```

Without ACCESS_TOKEN, the script skips API queries and generates SVGs with placeholder zeros.
